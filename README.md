
## Run Locally

Clone the project

```bash
 docker run -d --hostname rabbitmq --name myrabbitmq -p 15672:15672 -p 5672:5672 rabbitmq:management     
```




Install dependencies

```bash
  yarn 
```

Start the server

```bash
  yarn  start:dev 
```
